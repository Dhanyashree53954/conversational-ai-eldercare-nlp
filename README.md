# Conversational Agents for Eldercare Assistance using NLP
### MSc Major Project — University of Hertfordshire (2025/26)

## Overview
An intelligent AI dialogue system designed to support elderly users through natural, empathetic conversation. Built using transformer-based large language models (LLMs) with a focus on context retention, intent recognition, and accessible human-agent interaction.

## Key Features
- Intent classification and dialogue state management
- Context-aware, multi-turn conversation handling
- Empathy-focused response generation using LLMs
- Designed for accessibility and ease of use for elderly users

## Technologies Used
`Python` `Transformer-based LLMs` `NLP` `Intent Classification` `Dialogue State Management` `HuggingFace`

## Results
The system successfully demonstrated context-aware, empathetic dialogue for eldercare scenarios. Full evaluation metrics are included in the project report.
